Задание состоит из двух программ, которые необходимо реализовать. Взаимодействие программ должно быть реализовано через использование сокетов.
Программа-клиент. 
Должна состоять из двух потоков.
Поток 1.  Принимает строку, которую введет пользователь.   Должна быть проверка, что строка состоит только из цифр и не превышает 64 символа. После проверки строка должна быть отсортирована по убыванию и все четные значения элементов заменены на латинские буквы «КВ». После данная строка передается в поток 2, текущий поток должен ожидать дальнейшего ввода пользователя.
Поток 2. Принимает данные от потока 1. Должен вывести полученные данные на экран, рассчитать общую сумму всех элементов, которые являются численными значениями.  Полученную сумму передать в Программу-сервер. После этого поток ожидает следующие данные.
Примечание по Программе-клиент: Взаимодействие потоков должно быть синхронизировано, поток 2 не должен создавать паразитной нагрузки в ожидании данных.
Программа-сервер. 
Ожидает данные от Программы-клиент. При получении данных должна вывести данные на экран и произвести анализ из скольки символов состоит переданное значение. Если оно больше 2-ух символов и если оно кратно 32 выводит сообщение о корректности данных, иначе выводится сообщение об ошибке. Далее программа продолжает ожидать данные.

Требования к присылаемым решениям.
Готовое задание должно быть передано ответным письмом в tar.gz или zip архиве.
Проект должен компилироваться под ОС Linux.
При написании задания необходимо использовать стандарт C++ 11/14. 
Для сборки программа не должна требовать внешних библиотек или нахождения определенных файлов в специфичном месте.
Проект должен компилироваться с использованием cmake и/или make и g++.
В папке с исходным кодом не должно быть мусора: неиспользуемых файлов исходных кодов или ресурсов, промежуточных файлов сборки и т.д.
Максимальное время на выполнение задания –  2 недели.
Будут отмечены.
Структурная организация кода по функциональным классам.
Реализация обработки возможных ошибок на всем пути выполнения программ.
